Cerita Saya Perjalanan Menuju Web Developer dan Data science

Dari rasa penasaran sederhana tentang bagaimana sebuah website berjalan dan bagaimana data bisa bercerita, saya mulai menekuni dunia web development dan Data science sebagai mahasiswa Informatika. Setiap baris kode dan setiap analisis data adalah langkah kecil menuju impian besar: menciptakan solusi digital yang cerdas, interaktif, dan bermakna.

Perjalanan Saya

Setiap langkah dalam perjalanan ini membentuk cara saya memandang teknologi dan pengembangan web

1

Awal Ketertarikan

Pertama kali melihat website dan data visualisasi, saya bertanya-tanya: “Bagaimana ini bisa dibuat dan bagaimana datanya diolah?” Rasa penasaran itulah yang memulai semuanya.

2021 - 2023 (SMA)
2

Memulai Aksi

Mulai mencoba HTML, CSS, dan Python dari tutorial online. Website pertama saya sederhana, dan saya juga membuat program kecil untuk mengolah data. Kebanggaan kecil ini memicu semangat belajar lebih dalam.

2023 - Awal kuliah
3

Kuliah Informatika

Memilih jurusan Informatika untuk memperluas wawasan. Saya mulai memahami konsep penting di balik pengembangan web modern dan analisis data, seperti algoritma, struktur data, dan logika pemrograman.

2023 - Sekarang
4

Fokus Memperdalam

Menggabungkan keahlian front-end (JavaScript, React, Tailwind) dengan data science (Python, Pandas, visualisasi data). Setiap proyek jadi latihan nyata untuk mengasah skill dan siap hadapi dunia kerja.

2024 - Berkelanjutan

Filosofi Pengembangan Saya

Setiap baris kode bukan hanya soal fungsi, tapi juga menciptakan solusi yang berdampak—baik di web maupun data.

Kualitas di Atas Kecepatan

Saya memprioritaskan kode yang terstruktur dan mudah dipahami, serta analisis data yang jujur dan bernilai. Baik saat membangun UI/UX yang efisien atau mengolah data jadi wawasan, prinsip utamanya tetap sama: kualitas lebih penting dari kecepatan.

User Experience First

Teknologi hanyalah alat — yang utama adalah pengalaman pengguna. Dalam pengembangan web, saya memastikan UI responsif dan intuitif. Dalam data science, visualisasi harus mudah dipahami dan memberi makna.Setiap desain atau analisis harus mendukung pengguna, bukan membingungkan mereka.

Belajar Berkelanjutan

Teknologi terus berubah. Saya rutin belajar hal baru, baik di pengembangan web seperti React dan Tailwind, maupun di data science seperti Python dan visualisasi data. Bagi saya, belajar adalah bagian penting dari jadi developer yang terus berkembang.

Kenapa Web Dev & Data Science?

Dari banyak hal di dunia teknologi, saya paling tertarik pada web development dan data science karena keduanya seru dan saling melengkapi.

Bisa Bikin & Bisa Ngolah

Dengan web dev, saya bisa bikin tampilan website yang menarik dan mudah dipakai. Dengan data science, saya bisa mengolah data supaya lebih mudah dipahami.
