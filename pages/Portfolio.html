Challenge

Tantangan yang Dihadapi

Membuat form pendaftaran dengan tampilan elegan dan validasi real-time yang menjaga pengalaman pengguna tetap lancar dan responsif.

Approach

Pendekatan Solusi

Menggunakan HTML5 dan CSS3 untuk struktur dan desain responsif, serta JavaScript untuk validasi form secara real-time dan notifikasi dinamis.

Lessons Learned

Pembelajaran yang Didapat

Pentingnya memberikan feedback instan kepada pengguna, menjaga performa tetap optimal meskipun terdapat interaksi JavaScript secara dinamis.

// Validasi Username
  function checkUsername() {
    const value = username.value.trim();
    const small = username.nextElementSibling;
    if (value === '') {
      small.textContent = 'Nama wajib diisi';
      small.style.visibility = 'visible';
      return false;
    }
    small.style.visibility = 'hidden';
    return true;
  }
